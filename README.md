# Placement Tracker

Placement Tracker is a responsive web application designed to help students organize and monitor their job applications during placement season. It provides an easy way to track companies, job roles, application status, interview progress, and other important details.

## Features

- User Authentication (Sign Up & Login)
- Add Placement Applications
- Edit Existing Applications
- Delete Applications
- Search Companies
- Filter by Application Status
- Dashboard Statistics
- Dark / Light Mode
- Responsive Design
- Local Storage Support

## Dashboard

The dashboard displays:

- Total Applications
- Applied
- Online Assessment
- Interview
- Selected
- Rejected

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage

## How to Run

1. Clone the repository.

```
git clone <repository-url>
```

2. Open the project folder.

3. Run `index.html` using any modern browser.

No setup or installation is required.
